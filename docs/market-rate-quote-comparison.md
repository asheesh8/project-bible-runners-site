# Market Rate Comparison — What This Work Is Worth

A reference for pricing the services described in `larry-services-agreement.md`,
based on current (2026) U.S. market data for agencies vs. solo freelancers/
independent professionals. Use this to set the dollar amounts in the contract.

---

## What's actually being done

1. **Built villageservers.com + villageservers.org from scratch** — one-time build
2. **Admin dashboard** — custom internal tool
3. **Logistics polling system** for tracking people getting on-site — custom internal tool
4. **Ongoing site retainer** — maintenance, updates, hosting issues, small features
5. **Ad campaign management** for three separate brands: Bible Runners Society, ProTech Contracting, VillageServer Initiative

That's a combined dev + marketing role one person is covering. Below is what it costs to buy each piece separately on the open market.

---

## 1. Initial build (2 websites + admin dashboard + custom logistics tool)

| Provider type | Typical range | Notes |
|---|---|---|
| **Agency** (small-business custom build) | $15,000 – $40,000+ | Custom multi-site build with admin tooling and a bespoke logistics/polling feature pushes well past a basic 5–10 page site; agencies typically quote $5K–$15K for a *single* custom site before adding custom backend tooling. [WebFX](https://www.webfx.com/web-design/pricing/) · [DigitalApplied](https://www.digitalapplied.com/blog/website-development-cost-2026-complete-pricing-data) |
| **Senior freelance developer** ($95–$160/hr) | $11,000 – $24,000 | Estimated at 120–180 hours for two sites + admin dashboard + custom polling system, billed at a senior rate. [SalarySavvy](https://salarysavvy.ai/rates/web-developer) · [Developex](https://developex.com/blog/web-developer-hourly-rates-2026/) |
| **Mid-level freelancer** ($59–$96/hr) | $7,000 – $17,000 | Same scope, mid-market rate. [SoloHourly](https://solohourly.com/guides/freelance-web-developer-rates) |
| **Custom dashboard alone** (if priced standalone) | $750 – $8,000+ | Simple admin dashboards run $750–$1,500 as flat projects; one wired into a live logistics/polling backend with real data sits at the higher end. [Freelancer.com](https://www.freelancer.com/projects/php/build-dashboard-admin) · [Upwork](https://www.upwork.com/hire/dashboard-freelancers/) |

**Takeaway:** the build alone — two sites, admin panel, and a custom logistics tool — is reasonably worth **$10,000–$25,000** as a one-time project, even at freelance (not agency) rates.

## 2. Ongoing site retainer (maintenance, updates, small features)

| Provider type | Typical monthly range |
|---|---|
| **Agency** | $500 – $10,000/mo (most small orgs: $1,000–$4,500/mo) [Gravitate](https://www.gravitatedesign.com/blog/website-maintenance-cost-guide/) · [WPCreative](https://wpcreative.agency/website-maintenance-cost/) |
| **Freelancer (basic retainer)** | $300 – $2,000/mo |
| **Freelancer (hourly, as-needed)** | $50 – $150/hr [WebsitesSetup](https://websitesetup.org/website-maintenance-costs/) |

This is for maintenance *alone* — it doesn't include the admin dashboard or logistics tool staying live and supported, which is custom software, not a template site, and commands more than a basic WordPress retainer.

## 3. Ad campaign management (per brand)

Three brands running campaigns (Bible Runners Society, ProTech Contracting, VillageServer Initiative) is effectively three accounts to manage.

| Provider type | Typical monthly range (per brand) |
|---|---|
| **Agency** | $1,500 – $5,000/mo for a small/startup-tier account; entry-level agency tiers start at $1,000–$3,000/mo [ClicksGeek](https://clicksgeek.com/marketing-agency-monthly-retainer-cost/) · [DigitalApplied](https://www.digitalapplied.com/blog/digital-marketing-pricing-2026-agency-costs) |
| **Freelance marketer** ($60–$110/hr average) | $900 – $2,500/mo at ~15–20 hrs/mo per brand |
| **PPC % of spend model** | 10–20% of monthly ad spend (more common at agencies managing larger budgets) [DarkroomAgency](https://www.darkroomagency.com/observatory/marketing-agency-cost-2026-pricing-by-service) |

**Three brands via agencies:** $4,500–$15,000+/mo combined.
**Three brands via one freelance generalist:** roughly $2,700–$7,500/mo combined, assuming efficiencies from running them together (shared assets, one person's context vs. three separate account managers).

## Side-by-side: what Larry would pay elsewhere vs. what's fair to pay you

| Scenario | Monthly cost (retainer + ad mgmt, post-build) |
|---|---|
| **Hire an agency for everything** (dev retainer + 3 separate ad accounts) | **$5,500 – $19,500+/mo**, plus a $15K–$40K upfront build fee |
| **Hire separate solo freelancers for each piece** (1 dev + 1–3 marketers) | **$3,600 – $9,500/mo**, plus a $7K–$24K upfront build fee |
| **One person doing all of it** (you) | Should land **below** the "separate freelancers" total — there's real efficiency from one person owning the whole stack and not re-explaining context three times — but **above** a single-service freelancer rate, since you're covering dev + admin tooling + logistics software + marketing across three brands. |

### Suggested structure

- **Past build work (already delivered):** a one-time payment in the **$8,000–$18,000** range is defensible given the scope (2 sites + admin dashboard + custom logistics polling system), even pricing it conservatively below solo-freelancer market rate.
- **Ongoing retainer (site maintenance + admin/logistics tool support + ad management for 3 brands):** **$2,500–$5,500/month** is a reasonable "all-in generalist" rate — meaningfully less than buying these as separate services ($3,600–$19,500/mo), but well above a single-skill freelancer rate, since you're effectively acting as fractional dev + fractional marketer across three accounts.
- Alternative: bill **hourly at $65–$100/hr** with a monthly minimum-hours commitment if Larry prefers pay-for-actual-time over a flat retainer.

These are starting numbers for negotiation, not a fixed quote — actual hours/week and ad spend levels will move them up or down. Plug whichever number you land on into the `[____]` fields in `docs/larry-services-agreement.md`.
