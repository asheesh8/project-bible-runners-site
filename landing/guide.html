Board brief · What to set up first

Setup Guide

The setup guide gives field teams a plain order of operations so they can start with the part in front of them.

Plain answer

A good setup guide does not start with theory. It starts with a choice: phone, Raspberry Pi, microSD, USB, solar, projector, charging, or satellite.

Phone + Wi‑Fi

Connect one phone and confirm the local library opens.

Raspberry Pi

Power the server, wait for the Wi‑Fi network, and test the library from a phone.

Power

Confirm the source first: wall power, battery, solar, generator, or hybrid.

Group media

Only add projector, speakers, or satellite after the local library works.
