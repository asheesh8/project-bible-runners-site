Board brief - basic setup flow

Setup Guide

Unpack, power on, connect, test one file, and teach the next person. The field setup works in six consistent steps that any local leader can follow and repeat.

Plain answer

A setup guide should feel like a checklist, not an engineering manual. If a local leader cannot follow it under pressure, it is too complicated.

Field steps

Use it this way

  1. Unpack the kitPlace the case, server, cables, battery, printed guide, and optional projector/satellite items where they are visible.
  2. Connect powerUse wall power, battery, or solar-supported power based on the setting.
  3. Start the serverPower the Raspberry Pi and wait for the VillageServer Wi-Fi network.
  4. Join from a phoneConnect to the local Wi-Fi and open the library address or prompt.
  5. Test one resourceOpen one text, audio, video, or PDF file before inviting others.
  6. Pack consistentlyReturn every cable, adapter, card, and printed guide to the same labeled place.

Minimum setup

Power, server, phone, and one test file.

Optional setup

Projector, speaker, solar, satellite, USB, or microSD.

Best habit

Test one file before teaching the group.

Board takeaway

The guide turns the kit from equipment into a repeatable process.
