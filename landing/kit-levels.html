Board brief - planning a kit

Kit Levels and Costs

From a simple phone-sharing kit to a full satellite-enabled setup — choose the level that fits the field setting, budget, and mission need.

Plain answer

Start with the smallest kit that solves the field problem. Add power, projector, or satellite only when the mission setting actually calls for it.

Field steps

Use it this way

  1. Define the use caseAsk whether the need is personal phone sharing, group teaching, local library access, remote updates, or all of those.
  2. Pick the base levelStart with phone-sharing materials or a Raspberry Pi local library kit.
  3. Add power only if neededChoose battery or solar based on runtime and grid reliability.
  4. Add media for groupsUse projector and audio when one resource must serve a room or gathering.
  5. Quote satellite separatelyTreat satellite as an optional add-on because hardware, subscription, and power needs vary.
  6. Record item pricesKeep chips, Raspberry Pi, projector, solar, and satellite kit costs itemized.
