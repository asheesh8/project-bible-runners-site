Board brief - access paths

Get and Share Resources

People can access resources through local Wi-Fi, microSD, USB, phone-to-phone transfer, printed guides, or a prepared field kit.

Plain answer

The best access path is the one the local user can repeat without you standing there. Start simple, then add tools only if they help.

Field steps

Use it this way

  1. Choose the pathPick local Wi-Fi, phone sharing, microSD, USB, projector, or satellite based on the situation.
  2. Open one resourceDo not begin with a whole library. Open one Bible, audio file, film, or PDF first.
  3. Save it locallyPut the file somewhere the person can find again without data.
  4. Share nearbyUse AirDrop, Quick Share, LocalSend, USB, or microSD to move it to another device.
  5. Leave a handoutGive the matching PDF so the person can repeat the steps later.
  6. Ask them to teach itHave the new user show another person how to access the same file.

Local Wi-Fi

Best when many nearby phones need a library at once.

microSD or USB

Best for copying large folders or moving files without internet.

Phone to phone

Best when one person already has the resource.

Printed steps

Best for keeping the process repeatable after the team leaves.
