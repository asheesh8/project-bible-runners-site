Board brief - local Wi-Fi library

Raspberry Pi VillageServer System

The Raspberry Pi is the small local server. It stores the library and creates the local Wi-Fi experience people use from their phones.

Plain answer

Think of it as a small bookshelf with Wi-Fi. The phone connects to the bookshelf nearby instead of reaching out to the internet.

Field steps

Use it this way

  1. Power the serverConnect the Raspberry Pi to the approved battery, wall power, or solar-supported power source.
  2. Wait for the Wi-Fi nameGive the server time to start and broadcast the VillageServer network.
  3. Join from a phoneOpen Wi-Fi settings and connect to the VillageServer network. Mobile data is not required.
  4. Open the libraryUse the local address or captive portal shown with the kit.
  5. Test one fileOpen a Bible text, audio file, video, or PDF and confirm it loads on the phone.
  6. Share what helpsSave useful files and pass them to another phone when needed.

What it is

A small computer configured as an offline resource library.

What it is not

It is not the public internet and does not require a data plan.

Best use

Groups of nearby phones that need access at the same location.

Board takeaway

The Raspberry Pi makes the library feel like joining Wi-Fi and opening a website.
