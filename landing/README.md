# VillageServer informational site

`index.html` is the short guided front door: understand the initiative, check kit availability, choose a setup, and see the whole system flow.

`js/i18n.js` provides persistent full-homepage localization for seven languages, including translated availability results and component previews.

`initiative.html` contains the mission, kit model, pilots, and four-phase rollout. `access.html` contains Wi-Fi, microSD, USB, phone storage, and sharing guidance.

`guide.html?part=...` is the focused field-manual view. The homepage launcher and kit diagram link to eight component-specific walkthroughs.

The public landing experience is education-first: explain the initiative, show each technology path, and route visitors to setup/access guides. Donation checkout pages are intentionally not part of the primary public bundle.

All images are local WebP assets so the site remains resilient and lightweight.
