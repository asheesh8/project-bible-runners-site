Transfer Resources Between Devices

One route, one test file, then confirm it opens

Every transfer works best when you pick one route for the two devices in front of you, send one small file first, and confirm the receiving person can open it and find it again without help.

The most common transfer mistake is choosing the wrong method for the devices in the room, or sending too much too fast. Choose one route that matches the actual devices, test with one file, and only move the full library once you know the connection is solid.

Match the route to the devices

  • iPhone to iPhone — AirDrop when both are nearby and Bluetooth/Wi-Fi are on; confirm receiving is set to "Everyone" before starting
  • Android to Android — Quick Share when available, or LocalSend when both phones are on the same local network
  • iPhone and Android mixed — LocalSend on a shared local network, USB through a computer, or a microSD card with an adapter
  • Computer to phone — use Finder (Mac) or Windows File Explorer via USB cable; for Android, enable file transfer mode when the cable connects
  • microSD to phone — insert the card, open Files or a file manager, copy to internal storage, then confirm the file opens before removing the card

Running the transfer step by step

  1. Identify both devices

    Ask what each device is: iPhone, Android, Windows computer, Mac, USB drive, or microSD card. Getting this wrong means choosing the wrong method.

  2. Pick the matching route

    Use the list above to select the right method. Use AirDrop, Quick Share, LocalSend, USB cable, or card reader — whichever fits these two specific devices.

  3. Send one small file first

    Test with one PDF or audio file before sending a large folder. A test file reveals connection problems before wasting time on a large transfer that fails halfway.

  4. Confirm where the file landed

    Show the receiving person exactly where the file went: Downloads, Files, Photos, or a specific folder. If they can't find it, it won't get used.

  5. Open it offline

    Turn off mobile data or enable airplane mode if safe, then confirm the file still opens without a connection. This is the moment that proves the transfer worked.

  6. Let the receiver do the second transfer

    Have the receiving person do the next transfer to a third device while you watch. This is how the method becomes something they own, not something they were shown.
