Board brief - device routes

Transfer Resources Between Devices

Use the simplest transfer route for the devices in front of you: iPhone to iPhone, Android to Android, iPhone to Android, computer to phone, or microSD to phone.

Plain answer

Transfers work best when the helper chooses one route, sends one file first, and confirms the receiving person can open it without help.

Field steps

Use it this way

  1. Identify both devicesAsk what each device is: iPhone, Android, Windows computer, Mac, USB drive, or microSD card.
  2. Pick the matching routeUse AirDrop, Quick Share, LocalSend, USB cable, cloud-free local network, or card reader.
  3. Send one small fileTest with one PDF or audio file before sending a large folder.
  4. Find the saved fileShow the person where it landed: Downloads, Files, Photos, or a chosen folder.
  5. Open it offlineTurn off mobile data or airplane mode if safe, then confirm the file still opens.
  6. Repeat with the person watchingLet the receiver do the second transfer so they remember it.

iPhone to iPhone

Use AirDrop when both devices are nearby and Bluetooth/Wi-Fi are on.

Android to Android

Use Quick Share when available, or LocalSend on the same network.

Mixed phones

Use LocalSend, USB through a computer, or a microSD/card-reader path.

Computer to phone

Use Finder, Apple Devices, Android File Transfer alternatives, USB, or card readers.
