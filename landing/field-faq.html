Board brief - common questions

Field FAQ

The FAQ answers common field questions in plain language: internet, apps, power, privacy, content updates, and sharing.

Plain answer

When someone is confused, give them the next action, not a lecture. Each answer should point back to one simple step.

Field steps

Use it this way

  1. Listen for the real blockerIs the problem power, Wi-Fi, file location, device type, language, or confidence?
  2. Answer in one sentenceGive the shortest accurate answer first.
  3. Show the next actionPoint to the button, cable, Wi-Fi name, folder, or handout they need.
  4. Confirm on the deviceHave the person do the step while you watch.
  5. Write down repeatsIf several people ask the same question, update the guide or PDF.

Does it need internet?

No. The local library should work offline. Internet or satellite is optional for updates.

Does every phone need an app?

No for basic library access. Some sharing routes use common apps such as LocalSend.

What if power fails?

Use battery, solar, or local power planning. Keep the library useful even when the live link is off.

Can local leaders run this without tech skills?

Yes. The guides and PDFs are written for non-technical users who need to follow clear steps under field pressure.
