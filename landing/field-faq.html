Basics

Start here when someone is confused.

01Does VillageServer need internet?+

No. The core library is local. A phone connects to the VillageServer Wi‑Fi and opens the library nearby. Satellite or mobile data can help with updates, but the local library should still work without it.

02Why does my phone say “No internet”?+

That message is expected. The phone is connected to a local library, not the public internet. Choose to stay connected, then open the local address printed with the kit.

03Can people keep the files?+

Yes, if the resource is provided for saving. Open the file, use Download or Save, then test it in airplane mode before leaving the kit.

Devices

04What is easiest for iPhone to iPhone?+

AirDrop is usually the easiest Apple-only method. If it fails, use LocalSend or a computer bridge.

05What is easiest for Android to Android?+

Quick Share is usually the easiest Android-only method. Keep both phones awake and accept the receiving prompt.

06What is easiest between iPhone and Android?+

LocalSend is the cleanest cross-platform app when both devices are on the same local network or hotspot.

Problems

Most field fixes are simple checks.

Wi‑Fi

Phone cannot see the network

Move closer, wait for the server to finish starting, then check if another phone can see it.

File

File will not open

Try one small file first, confirm it finished downloading, and check whether the right app is installed.

Power

Kit keeps shutting off

Reduce optional loads, confirm the approved power cable, and check solar or battery charge before restarting.
